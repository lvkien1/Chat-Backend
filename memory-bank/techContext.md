# Công nghệ sử dụng

## Core Stack
- .NET 6+
- ASP.NET Core Web API
- SignalR (WebSocket)

## Database
- MySQL 8.0+
- Dapper (Micro ORM)
- Dapper.Contrib
- MySqlConnector (high-performance driver)

## Authentication
- JWT Bearer Token
- ASP.NET Core Identity (option)

## Packages chính:
- Microsoft.AspNetCore.SignalR
- Dapper
- Dapper.Contrib
- MySqlConnector
- Microsoft.AspNetCore.Authentication.JwtBearer

## Development Tools
- Swagger/OpenAPI
- Postman
- MySQL Workbench
